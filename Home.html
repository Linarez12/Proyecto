INTRODUCCIÓN AL HTML

¿Que es el Html?

HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el estándar fundamental para crear y estructurar páginas web. Desde su aparición en la década de 1990, HTML ha evolucionado significativamente, permitiendo a los desarrolladores web crear sitios web más complejos y dinámicos. Este texto busca proporcionar una introducción completa a HTML para aquellos que recién comienzan en el mundo del desarrollo web. HTML es un lenguaje de marcado, lo que significa que utiliza "etiquetas" para definir la estructura y el contenido de un documento. Estas etiquetas no son visibles en la página web final, sino que son interpretadas por los navegadores para renderizar el contenido en una forma visualmente comprensible para los usuarios.

Historia del Html

HTML (HyperText Markup Language) es el lenguaje de marcado fundamental para la creación de páginas web, desarrollado por Tim Berners-Lee en 1991 mientras trabajaba en CERN, el laboratorio europeo de física de partículas. Su objetivo inicial era facilitar el intercambio de documentos científicos mediante un formato estandarizado que pudiera ser visualizado en cualquier computadora. La primera versión pública de HTML, conocida como HTML 2.0, se lanzó en 1995 bajo la supervisión del IETF (Internet Engineering Task Force), estandarizando muchas características que los navegadores ya utilizaban de manera no oficial. Posteriormente, el W3C (World Wide Web Consortium) asumió la responsabilidad del desarrollo y estandarización de HTML, lanzando HTML 3.2 en 1997 y HTML 4.0 en 1999, que introdujeron importantes mejoras en la presentación y estructura de las páginas web. Con la evolución de la web, HTML5 se desarrolló como una colaboración entre el W3C y WHATWG (Web Hypertext Application Technology Working Group), lanzándose oficialmente en 2014. HTML5 introdujo numerosas mejoras, incluyendo nuevas APIs, etiquetas semánticas y soporte multimedia, permitiendo una mayor interactividad y capacidades multimedia sin la necesidad de plugins externos. HTML sigue evolucionando, adaptándose a las necesidades de la web moderna y manteniéndose como la piedra angular del desarrollo web. Tim Berners-Lee y el nacimiento de la Web (1989-1991) HTML fue creado por Tim Berners-Lee, un científico del CERN (Conseil Européen pour la Recherche Nucléaire o European Organization for Nuclear Research). En 1989, Berners-Lee propuso un proyecto basado en el hipertexto para facilitar el intercambio de información entre investigadores. En 1990, escribió el primer navegador web, WorldWideWeb, y el primer servidor web, iniciando así la World Wide Web. El primer documento HTML fue escrito en 1991 por Berners-Lee y se tituló "HTML Tags". Este documento describía 18 etiquetas básicas que constituyen la base de HTML, muchas de las cuales siguen en uso hoy en día.
